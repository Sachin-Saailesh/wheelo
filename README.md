# 🚗 Wheelo

**Wheelo** is a modern, premium mobile application built with Flutter. It's designed specifically for the unique needs and dynamics of the targeted Indian market.

This project is cultivated with care and purpose for a friend who is embarking on an exciting entrepreneurial journey to build a robust business in the mobility and transportation sector in India.

## 🌟 What is Wheelo?

Wheelo is a comprehensive, dual-sided marketplace tailored specifically for the pre-owned vehicle rental sector. It aims to provide a seamless, reliable, and user-friendly platform where users have two flexible options:

1. **🌍 Rent a Vehicle:** Browse, view, and rent a wide variety of pre-owned vehicles listed in the market for convenient personal mobility.
2. **🤝 Become a Host:** Vehicle owners can seamlessly list their pre-owned cars or bikes on the platform, making them available for rental and generating an active income stream.

The application features a sleek, curvy, and modern UI crafted to offer a premium, "Google-quality" user experience. This design ensures that users in the Indian market have access to a top-tier digital service while navigating the peer-to-peer vehicle sharing and rental ecosystem.

### ✨ Key Features

- **Dual Marketplace Roles:** Effortlessly switch between being a renter looking for a reliable ride, or a host managing listed vehicles.
- **Modern Authentication:** Secure Phone/OTP-based user authentication powered by Firebase.
- **KYC Verification:** Built-in multi-step KYC verification flow to ensure user trust, safety, and regulatory compliance.
- **Search & Booking:** Intuitive search interfaces, detailed overlays, and streamlined booking screens.
- **Premium UI/UX:** A consistent, sky-blue and white theme focusing on clean aesthetics, dynamic animations, and effortless navigation.
- **Robust Backend:** Integrated with Firebase Authentication and Cloud Firestore for real-time data synchronization and user management.

## 🚀 Getting Started

To set up this project locally:

1. **Ensure Flutter is installed:**
   Follow the [official guide](https://docs.flutter.dev/get-started/install) to install Flutter if you haven't already.

2. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd wheelo
   ```

3. **Install Dependencies:**

   ```bash
   flutter pub get
   ```

4. **Run the App:**
   ```bash
   flutter run
   ```

_(Note: Ensure you have your Firebase configuration files — `google-services.json` for Android and `GoogleService-Info.plist` for iOS — placed in their respective directories before building the app.)_
